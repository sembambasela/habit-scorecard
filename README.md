# habit-scorecard
A desktop habit tracking application inspired by Atomic Habits built with Electron, React, and SQLite.

## Features

- Track daily habits
- Classify habits as Positive, Neutral, or Negative
- View streaks and completion rates
- Visualize progress with charts
- Store data locally using SQLite

## Tech Stack

- Electron
- React
- Vite
- SQLite
- Tailwind CSS
